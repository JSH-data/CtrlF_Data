## 점근표기법 이란?!
### 점근표기법
    어떤 함수의 증가 양상을 다른 함수와의 비교로 표현하는 수론과 해석학의 방법이다.[위키백과]

즉, CS에서는 알고리즘의 점근적 효율성(입력크기가 극한으로 증가할 때 알고리즘의 수행시간이 어떻게 증가하는지)에 관심을 두고 자연수들의 집합으로 하는 함수들로 <u>__알고리즘의 수행시간을 나타내기 위해서__</u> 사용한다.
### 점근표기법의 종류
- 대문자 O 표기법
- 소문자 o 표기법
- 대문자 오메가(Ω) 표기법
- 소문자 오메가(ω) 표기법
- 대문자 세타(Θ) 표기법
